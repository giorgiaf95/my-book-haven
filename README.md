# Bixblion

Piattaforma web per lettori: libreria personale, recensioni, community e sfide.

## Requisiti

- Node.js 18+
- npm 9+

## Avvio locale

```bash
npm install
npm run dev
```

App disponibile su `http://localhost:8080`.

## Build produzione

```bash
npm run build
npm run preview
```

## Test

```bash
npm run test
```
